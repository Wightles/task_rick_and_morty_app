# task_rick_and_morty_app

тестовое задание для Effective Mobile

Проект: Rick and Morty Characters App
Краткое описание:
Мобильное приложение на Flutter для просмотра персонажей мультсериала "Рик и Морти" с возможностью добавления в избранное и полноценной офлайн-работой.

Архитектура и технологии:

State Management:
BLoC (Cubit) - для управления состоянием персонажей и избранных
Provider - для управления темой и навигацией

Локальное хранилище:
SharedPreferences - для хранения ID избранных персонажей

Файловый кэш - для изображений и данных персонажей

Работа с сетью:
Dio - для HTTP-запросов к Rick and Morty API

REST API - стандартный подход с пагинацией

Основные экраны:
1. Главный экран (Персонажи):
Отображение: Карточки персонажей с изображением, именем, статусом, видом и локацией

Интерактивность:
Кнопка "звездочка" для добавления/удаления из избранного
Пагинация при скролле (автоматическая загрузка новых страниц)
Особенности: Автоматическое кэширование всех просмотренных страниц

2. Экран "Избранные":
Отображение: Карточки избранных персонажей
Сортировка: По имени (А-Я/Я-А), статусу, виду

Управление:
Удаление свайпом (с вибрацией и анимацией)
Кнопка сортировки в виде плавающей кнопки
Пустой экран: Анимированная смена иконки (смайлик ↔ звезда)

3. Экран "Настройки":
Управление кэшем: Просмотр статистики и очистка данных
Обновление данных: Кнопка для принудительного обновления с сервера
Переключение темы: Светлая/темная тема

Ключевые функции:
Реализовано:
Полная офлайн-работа - все просмотренные данные сохраняются локально
Интеллектуальное кэширование:
Данные персонажей (все просмотренные страницы)
Изображения (сохранение в файловой системе)
Автоматическая инвалидация через 24 часа
Синхронизация избранных - мгновенное обновление между экранами
Тактильная обратная связь - вибрация при действиях с избранным

Анимации:
Переход между смайликом и иконкой на пустом экране
Анимация карточек при появлении
Эффекты при добавлении/удалении избранного

Дизайн и темы:
Адаптивный UI - под разные размеры экранов
Темная/светлая тема с автоматическим переключением
Собственная цветовая палитра в AppThemes
Плавные переходы и современный Material Design
